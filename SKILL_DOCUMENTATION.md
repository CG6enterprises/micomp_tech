# Micomp_Tech - Skill for Claude, Gemini, and ChatGPT

## Overview

Mincomp_Tech is a comprehensive educational and freelance platform designed for statisticians and data professionals. This skill enables AI assistants (Claude, Gemini, ChatGPT) to:

1. **Teach Statistical Sciences** - Guide students through data collection, analysis, and interpretation
2. **Manage Freelance Services** - Track projects, clients, and billing
3. **Provide Analysis Support** - Offer statistical insights and recommendations
4. **Generate Learning Materials** - Create exercises, explanations, and course content

## Technology Stack

- **Frontend:** HTML5, CSS3 (Blue & Gold theme), JavaScript
- **Backend:** Python Flask
- **Database:** SQLite (PostgreSQL ready)
- **AI Integration:** Claude API, Google Gemini API, OpenAI API
- **Libraries:** NumPy, Pandas, SciPy for analysis

## Quick Start

### Installation
```bash
git clone https://github.com/CG6enterprises/micomp-tech.git
cd micomp-tech
pip install -r backend/requirements.txt
python backend/app.py
```

### Access
- **Web:** http://localhost:5000
- **API:** http://localhost:5000/api

### Configuration
1. Copy `.env.example` to `.env`
2. Add your AI API keys (Claude, Gemini, ChatGPT)
3. Configure database if needed
4. Run the application
