# Micomp_Tech: Statistical Sciences & Data Management Platform

A comprehensive web platform for teaching statistical sciences, data management, and freelance data analysis services.

## Features

### 📚 Teaching Platform
- **4 Complete Courses:**
  - Data Collection 101
  - Statistical Basics
  - Data Processing
  - Advanced Analysis

- **Interactive Learning:**
  - Lecture notes and tutorials
  - Interactive quizzes with auto-grading
  - Sample datasets (Business, Education, Taxes)
  - Step-by-step data collection guides
  - Certificates upon completion

- **Statistical Analysis Tools:**
  - Descriptive statistics calculator
  - Data visualization
  - Hypothesis testing
  - Correlation & regression analysis

- **AI Assistant:**
  - AI-powered chatbot for student questions
  - Automated concept explanations
  - Claude, Gemini, ChatGPT integration

### 💼 Freelance Services Hub
- **Project Portfolio:**
  - Showcase completed projects
  - Client testimonials
  - Case studies

- **Client Management:**
  - Data collection forms
  - Project tracking dashboard
  - Billing & invoice system
  - Reusable analysis templates

## Tech Stack
- **Frontend:** HTML5, CSS3 (Professional Blue & Gold Theme), JavaScript
- **Backend:** Python Flask/FastAPI
- **Database:** SQLite (with migration to PostgreSQL ready)
- **AI Integration:** Claude API, Google Gemini API, OpenAI API

## Getting Started

### Installation
```bash
git clone https://github.com/CG6enterprises/micomp_tech.git
cd micomp_tech
pip install -r backend/requirements.txt
python backend/app.py
```

Visit `http://localhost:5000` in your browser.
